# fan-get-money 使用说明

## 六大命令

| 命令 | 做什么 | 什么时候用 |
|------|--------|------------|
| `搞钱初始化` | 5 个问题建立画像，判定 T0-T3 | 第一次用 |
| `帮我找机会` | 搜真实数据，输出机会列表 | 想找方向 |
| `验证 XX` | 过五层反诈，判高危/存疑/可行 | 不确定靠不靠谱 |
| `给我行动方案` | 生成两周具体计划+止损线 | 选定方向后 |
| `复盘` | 对比预测 vs 实际 | 做了一段时间 |
| `搞钱状态` | 查看画像、进度、下一步 | 随时 |

## 完整流程

```
搞钱初始化 → 帮我找机会 → 验证 XX → 给我行动方案 → 执行... → 复盘 → 搞钱状态
```

## 各平台触发方式

| 平台 | 怎么触发 |
|------|----------|
| Codex | `/fan-money-init` 等斜杠命令 |
| Claude Code / Gemini / Hermes / WordBuddy | 直接说中文，如"搞钱初始化" |
| Cursor / Copilot / Windsurf | 安装后自动生效，说中文即可 |

## 适配器

两个数据抓取工具，读取真实成交和招聘数据：

```bash
# 闲鱼（一手成交数据）
cd adapters/xianyu && npm install
./launch-chrome.sh        # 扫码登录闲鱼
node read-xianyu.mjs "AI头像" 9222

# BOSS直聘（一手招聘数据）
cd adapters/boss          # 零依赖
./launch-chrome.sh        # 扫码登录BOSS
node read-boss.mjs "AIGC" 100010000 9222
```

> 只读，不自动登录，不批量爬取。

## T0-T3 分层

| 层级 | 你是什么状态 | 适合方向 |
|------|-------------|----------|
| T0 | 零技能/零资金 | 平台任务、数据标注 |
| T1 | 会用 AI 工具 | 接单类技能服务 |
| T2 | 精通 AI + 有行业经验 | B端交付、出海、SaaS |
| T3 | 有团队/资本 | 产品化、规模化 |

## 反诈速查

看到这些词直接警惕：稳赚、躺赚、日入、包赚、月入过万、日结、宝妈、小白、保证收益、零风险

碰到以下情况直接跑：先交钱、加 QQ/微信私聊、试稿不给钱、截图造假
